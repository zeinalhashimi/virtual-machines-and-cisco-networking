# IT Platforms Project – Virtual Machines and Cisco Networking

## Project Overview

This project was developed as part of the IT Platforms course.  
The aim of the project is to demonstrate practical knowledge of virtual machines, networking concepts, and platform-based application deployment.

The project combines virtual machine setup, Cisco networking concepts, and a web-based application to show how modern IT platforms are designed, deployed, and managed.

## Objectives

- Understand and configure virtual machines
- Apply basic Cisco networking concepts
- Develop and deploy a simple web application
- Use GitHub for version control and project management
- Follow platform best practices such as environment variables

## Technologies Used

- Virtual Machines
- Cisco Networking
- Python
- Flask
- HTML/CSS
- Git & GitHub

## Project Structure

- `app.py` – Flask backend application
- `templates/` – HTML files used by Flask
- `static/` – CSS and static assets
- Documentation files
